## ✨ Features

- 🔐 Authentication system (Login / Signup)
- 📩 Send and manage messages
- ⚡ Blazing fast performance with Next.js
- 🔄 API routes (backend inside Next.js)
- 📱 Fully responsive design
- 🎨 Modern UI/UX

---

## 🧠 Built With

- ⚡ Next.js (App Router)
- ⚛️ React.js
- 🎨 Tailwind CSS 
- 🛢️ MongoDB
- 🔐 JWT Authentication
- ☁️ Vercel Deployment
